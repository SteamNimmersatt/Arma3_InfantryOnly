# Arma3_InfantryOnly
Arma3 mod which enforces infantry-only gameplay

## Features
This mod disables all weapons on vehicles, forcing infantry-only gameplay while still allowing vehicles for transportation.

Key features:

- **Multiplayer Compatible**: Works in both singleplayer and multiplayer environments
- **Complete Vehicle Weapon Disabling**: Removes ammunition from all vehicle weapons and turrets
- **Configurable Settings**: Adjustable through CBA settings interface
- **Event-Driven Processing**: Uses CBA event handlers for immediate response to vehicle spawning
- **Whitelist Support**: Configure specific vehicle types to retain their weapons
- **Advanced Vehicle Classification**: Configure specific vehicle categories to retain their weapons:
  - Static turrets
  - Technical vehicles (armed light wheeled vehicles)
  - APCs and IFVs
  - Tanks
  - Helicopters
  - Fixed-wing aircraft
  - UAVs/Drones
  - Naval vessels (boats and ships)


See [documentation/features.md](documentation/features.md) for a complete list of features.
